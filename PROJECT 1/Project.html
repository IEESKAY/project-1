TITLE: WEB DEVELOPMENT

Web development is the discipline of building and maintaining websites and web applications. At its core, it revolves around three pillars

  • HTML for structure
  • CSS for styling
  • JavaScript for interactivity

HTML

HTML (HyperText Multi-Link) is the foundational language for building web pages. It gives structure to everything you see on a website — text, images, forms, buttons, sections, tables, and so on.

CSS

CSS (Cascading Style Sheet) CSS is the language used to control how a webpage looks. It handles layout, colours, spacing, fonts, animations—everything visual.

JavaScript

JavaScript (JS) is the programming language that makes webpages interactive. If HTML is the structure and CSS is the design, JavaScript is the brain.

Web development ganerall revolves along these 3 pillars each intreracting together to form a website.
